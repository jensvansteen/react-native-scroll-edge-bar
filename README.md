# react-native-scroll-edge-bar

Scroll Edge for react native

## Installation


```sh
npm install react-native-scroll-edge-bar react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```


## Usage


```js
import { ScrollEdgeBarView } from "react-native-scroll-edge-bar";

// ...

<ScrollEdgeBarView color="tomato" />
```
